# Jingyan Lou's Portfolio Website  

## Overview  
This portfolio website showcases my work, skills, and projects as a full-stack developer, emphasizing clean design and functionality.  

### **Stack**  
- **Frontend**: React.js, TypeScript, TailwindCSS  
- **Design**: Swiss Design  

### **What is Swiss Design?**  
Swiss Design, also known as the International Typographic Style, is a graphic design style emphasizing cleanliness, readability, and objectivity. It often incorporates:  
- Grid-based layouts.  
- Sans-serif typography.  
- Minimalistic and structured designs.
